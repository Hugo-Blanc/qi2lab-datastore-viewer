## Installation

You can install `qi2lab-datastore-viewer` via [pip]:

```bash
pip install qi2lab-datastore-viewer
```

If napari is not already installed, you can install `qi2lab-datastore-viewer` with napari and Qt via:

```bash
pip install "qi2lab-datastore-viewer[all]"
```


To install latest development version:

```bash
pip install git+https://github.com/Hugo-Blanc/qi2lab-datastore-viewer.git
```
